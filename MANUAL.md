## 1. 快速上手

1. 双击 `RedStudio.exe` 启动
2. 点击右上角 **⚙ 设置**
3. 在「供应商」区域添加你的 API（见第 2 节）
4. 点击顶部下拉框选择模型，开始对话

配置文件保存在 `C:\Users\你的用户名\.aistory\config.json`，历史记录在同目录 `history.json`。

## 2. 供应商与模型配置

### 添加云端 API（DeepSeek / Qwen / 豆包等）

设置 → 供应商 → **+ 添加供应商**

| 字段 | 说明 |
|------|------|
| 名称 | 随便起，如"深度求索" |
| 类型 | 选「OpenAI 兼容」 |
| Base URL | 供应商 API 地址 |
| API Key | 你的密钥 |

常用地址：
- DeepSeek：`https://api.deepseek.com`
- 通义千问：`https://dashscope.aliyuncs.com/compatible-mode/v1`
- 豆包：`https://ark.volcengine.com/api/v3`

### 添加 Ollama 本地模型

| 字段 | 填写 |
|------|------|
| 类型 | 选「Ollama」 |
| Base URL | `http://localhost:11434`（默认） |
| API Key | 留空 |

添加后点击模型下拉旁边的 **刷新图标** 加载可用模型。

### 切换当前供应商

点击侧边栏顶部的供应商名称下拉。

## 3. 对话基础操作

| 操作 | 方式 |
|------|------|
| 发送消息 | `Enter` 或点击 ↑ 按钮 |
| 换行 | `Shift + Enter` |
| 停止生成 | 点击底部「停止生成」按钮 |
| 复制 AI 回复 | 回复右下角复制图标 |
| 朗读 AI 回复 | 回复右下角朗读图标 |
| 新建对话 | 侧边栏顶部「+ 新建对话」按钮 |

### 系统提示词

消息框上方有可折叠的系统提示词输入区。每个对话独立设置。可从提示词库快速填入（点击 📚 图标）。

### 作者注记

系统提示词下方的可折叠输入区。输入的内容会在每轮 API 调用时自动注入到 context 末尾附近，用于临时修正 AI 行为，无需反复修改系统提示词。

### Token 统计

输入框底部实时显示当前对话累计 token 消耗。侧边栏内嵌今日 token 仪表盘。

## 4. 思考模式

**适用模型：** DeepSeek-R1、Qwen3（支持 `enable_thinking` 参数的模型）

点击输入框旁的 **💡 按钮**（紫色高亮 = 已开启）。

开启后 AI 会输出推理过程，以折叠块「思考过程」显示，正文在其下方。

## 5. 联网搜索

**前提：** 需要 Ollama 账号 API Key（免费注册：https://ollama.com）

设置 → 联网搜索 → 填入 **Ollama API Key**

点击输入框旁的 **🌐 按钮**（蓝色高亮 = 已开启）。

开启后每次发送消息会自动调用 Ollama 搜索 API，将结果注入上下文。

## 6. JRPG 模式

类 P5R（女神异闻录5）的文字冒险模式。

### 角色创建

侧边栏模式切换 → 选「JRPG」→ 点击「⚔ 创建角色 / 开始冒险」。

| 字段 | 说明 |
|------|------|
| 角色名 | 必填，玩家主角名字 |
| 初始元素 | 9 选 1：🔥火 💧水 🌍地 🌪风 ❄️冰 ⚡雷 🌳木 ✨光 🌑暗 |
| 战斗属性 | 力/魔/速/防/体力，各 1-3，上限 100 |
| 社交属性 | 德行/智识/体魄/魅力，各 1-2，上限 50 |

面板底部显示已保存的角色库，点击可直接导入名字和元素。

### 战斗属性说明

| 属性 | 说明 |
|------|------|
| 力 | 物理攻击力，影响物理技能伤害 |
| 魔 | 魔法攻击力，影响魔法技能伤害 |
| 速 | 速度，影响行动顺序、闪避率、暴击率 |
| 防 | 防御力，影响伤害减免 |
| 体力 | HP 上限，影响生存能力 |

### 元素克制

10 种元素相互克制：克制 = 2 倍伤害，被克制 = 0.5 倍。

### 技能系统

- 每个元素有 13 个技能（5 初级 / 4 中级 / 3 高级 / 1 究极）
- 技能分三类：物理（受力加成）、魔法（受魔加成）、辅助（无伤害）
- 玩家最多装备 **4 个技能**，可随时切换
- 升级/剧情时可学习新技能，超出 4 个需替换
- 被替换的技能进入遗忘列表，可随时找回

### NPC 系统

- 每次新游戏随机生成 **5 位女 NPC**（名字/定位/性格/元素全部随机）
- NPC 首次登场时 AI 会输出角色卡，旁边有「📥 导入」按钮
- 导入后保存到角色库，下次新游戏可在创建面板选择占位
- 好感度 0-100，4 阶段（陌生人 → 朋友 → 暧昧 → 恋人）
- 支持后宫路线（多人同时达到"恋人"）或纯爱单推

### NPC 面板

点击状态栏 👥 按钮展开，每个 NPC 卡片显示：
- 头像 + 元素属性角标
- 名字 + 角色定位
- 性格标签
- 好感进度条 + 数值
- Rank 等级（1-4）+ 阶段名

## 7. 小说模式

互动写作模式，AI 扮演叙述者和所有角色。

### 角色库

点击侧边栏底部**📚 知识库** → 角色库打开。

- **小说角色 / 聊天角色** 两个 Tab 分开管理
- 点击角色行直接选用
- 支持 **导出/导入 JSON** 和 **SillyTavern PNG 导入**

女主角角色卡字段：姓名、出生日期、身高、身材、外貌、性格、语言风格、爱好、出身、当前状态、补充信息。

### 开始新故事

点击侧边栏「▶ 开始新故事」，弹出设定面板：

| 字段 | 说明 |
|------|------|
| 选择模板 | 内置"言情宿舍""体育竞技"等，可在设置中自定义 |
| 男主名 | 主角名字，注入系统提示词 |
| 叙事视角 | 第一人称 / 第三人称 |
| 字数/轮 | 每次 AI 输出约多少字 |

### 模板系统

模板是完整的 prompt，包含角色设定、输出格式、行为规则等。在设置面板中管理：

- 内置预置模板，支持自定义创建/编辑/删除
- 模板支持占位符：`{hero_name}` `{story_dir}` `{word_count}` `{pov}`

### 选项与好感度

AI 回复末尾的 `[CHOICES]` 块或 `A. B. C. D.` / `1. 2. 3. 4.` 格式渲染为可点击选项按钮。好感度完全由用户提示词定义，系统不自动注入规则。

## 8. TTS 朗读

### 引擎选择（设置 → 朗读）

| 引擎 | 说明 |
|------|------|
| 小米 MiMo TTS | 在线，限时免费，4 个中文声线，长文本自动分段并行下载 |
| Edge TTS | 在线，微软神经网络，28 个中文声线 |
| Windows SAPI | 离线，系统内置 |

**MiMo API Key 获取：** https://platform.xiaomimimo.com

### 使用

点击 AI 回复右下角 🔊 按钮开始朗读。底部出现「⏹ 停止朗读」按钮。

---

## 9. 历史对话管理

- 侧边栏列出所有历史对话，点击切换
- 顶部搜索框实时过滤历史标题
- 每个历史条目右侧有 ✎ 重命名（自定义弹窗）和 × 删除按钮
- 对话标题自动取第一条用户消息前 30 字

## 10. 界面操作

| 操作 | 方式 |
|------|------|
| 折叠/展开侧边栏 | 标题栏左侧 ‹ 按钮 |
| 切换主题 | 底部 🌙 按钮（5 种：亮色/护眼米/护眼绿/深蓝/纯黑） |
| 导出对话 | 底部 ↓ 按钮（Markdown 格式） |
| 拖动窗口 | 拖拽顶部标题栏空白区域 |
| 缩放窗口 | 拖拽窗口任意边或角 |
| 最大化 | 双击标题栏 |
| 压缩记忆 | 侧边栏底部「压缩记忆」按钮，将旧消息摘要释放上下文空间 |
